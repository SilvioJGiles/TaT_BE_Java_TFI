# TaT_BE_Java_TFI

## Descripción

Sistema de gestión desarrollado con **Java**, **Spring Boot** y **Maven** como backend, con una interfaz web en **HTML, CSS y JavaScript**.

La aplicación permite:

- Gestión de productos.
- Gestión de categorías.
- Registro de pedidos.
- Consulta del historial de pedidos.
- Administración del sistema.

## Requisitos

- Java 17 o superior.
- Maven.

## Ejecución

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/SilvioJGiles/TaT_BE_Java_TFI.git
   ```

2. Abrir el proyecto con el IDE de preferencia (IntelliJ IDEA, Eclipse o VS Code).

3. Ejecutar la clase:

   ```
   EcommerceApplication.java
   ```

4. Una vez iniciado Spring Boot, abrir un navegador y acceder a:

   ```
   http://localhost:8080/
   ```

## Menú principal

La aplicación cuenta con las siguientes opciones:

- Gestionar Productos
- Gestionar Categorías
- Realizar Pedido
- Historial de Pedidos
- Administración

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Maven
- HTML5
